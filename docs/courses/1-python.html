Copying a List

To get a new copy of a list, use the copy() method. If you just do newList = oldList, your new variable will just be a [reference](https://en.wikipedia.org/wiki/Reference\(computerscience\)) to the original list... which is not what we want. python nums = [4, 3, 2, 1] numsCopy = nums.copy() # numsCopy is [4, 3, 2, 1]
